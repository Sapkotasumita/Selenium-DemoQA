Selenium DemoQA Automation (Python)
Project Description
This project contains Selenium automation scripts written in Python to test different features of the DemoQA website.
It is created for learning and practicing UI automation testing on a real demo application.

Website Tested
https://demoqa.com/

Tools Used:
Python
Selenium WebDriver
Google Chrome
ChromeDriver
VS Code
Git & GitHub

Test Cases Covered:
Text Box
Buttons
Check Box
Radio Button
Links
Add Record
Delete Record
File Upload & Download
Swagger UI (basic testing)

How to Run:
Clone the repository
git clone https://github.com/Sapkotasumita/Selenium-DemoQA
Install Selenium
pip install selenium
Make sure ChromeDriver is installed and added to PATH
Run any test file
python Textbox.py

Screenshots:
Screenshots are saved in separate folders during test execution for validation and debugging.
